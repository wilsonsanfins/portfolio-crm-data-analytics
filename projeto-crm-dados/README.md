# Projeto de Inteligência de CRM

Este projeto automatiza a classificação de clientes de uma base de varejo.

## 🛠 Ferramentas
- Python (Pandas) para limpeza e segmentação.
- SQL para extração de métricas de negócio.

## 📈 O que este projeto faz:
1. Identifica clientes que não compram há mais de 90 dias (Risco de Churn).
2. Separa clientes de alto valor (VIP) para campanhas exclusivas.
3. Gera uma base pronta para o time de marketing disparar e-mails.
