# Pipeline de Dados: Performance de E-mail Marketing

Este projeto simula um fluxo real de CRM onde dados de diferentes fontes precisam ser unificados.

##  Objetivo
Calcular a taxa de engajamento das campanhas de Black Friday e Natal.

##  Tecnologias
 Python (Pandas) para o Merge (União) de tabelas.
 SQL para cálculo de conversão (CTR).

##  Insight de Negócio
Ao unir as bases, é possível identificar quais segmentos de clientes ignoram as campanhas, permitindo ajustar o conteúdo para evitar descadastros no Opt-out.
